
* 如果题目没有要求必须在原数组实现的，那么可以使用一个额外的数组将非0数放入其中，然后非0补全
* 但需要在原数组实现，加一个游标j在原数组做修改
```js
var moveZeroes = function(nums) {
    let j = 0
    let n = nums.length
    for(let i=0;i<n;i++){
        if(nums[i]!==0){
            nums[j]=nums[i]
            j+=1
        }
    }
    for(;j<n;j++){
        nums[j] = 0
    }
    return nums
};
```
